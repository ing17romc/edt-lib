# Recipes

The repository includes task-oriented recipes for common integration flows:

- login form composition
- dashboard shell composition
- generic forms
- Next.js App Router setup
- Vite integration
- Vercel deployment

These recipes exist in `docs/recipes/` and are intended to be used alongside component stories.
