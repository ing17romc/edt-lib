# `examples/vercel-ready`

Minimal Vite deployment seed for Vercel using `edt-lib`.

## Install

```bash
cd examples/vercel-ready
npm install
```

## Build

```bash
npm run build
```

This example demonstrates:

- package-root imports
- the raw Sass entry `edt-lib/index.scss`
- a simple `vercel.json` for static deployment
