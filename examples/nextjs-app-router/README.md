# `examples/nextjs-app-router`

Minimal Next.js App Router consumer for `edt-lib`.

## Install

```bash
cd examples/nextjs-app-router
npm install
```

## Run

```bash
npm run dev
```

This example demonstrates:

- `import 'edt-lib/styles.css'` in `app/layout.tsx`
- `ThemeProvider` at the root layout
- package-root imports only
