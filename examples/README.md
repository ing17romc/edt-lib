# Examples

This directory contains consumer-facing integration seeds for `edt-lib`.

- `vite-react`: recommended Vite integration using `edt-lib/styles.css`
- `nextjs-app-router`: App Router setup with `ThemeProvider`
- `vercel-ready`: Vite deployment seed with `vercel.json`

These examples are intentionally minimal and are meant to stay aligned with the package root API.
